# InterFlash Acapulco - Internet de Fibra Óptica

Este es el proyecto de la landing page oficial de **InterFlash Acapulco**, un proveedor líder de internet de fibra óptica en Acapulco, Guerrero.

## 🚀 Características
- **Diseño Premium**: Interfaz moderna, centrada y optimizada para la conversión.
- **Totalmente Responsivo**: Adaptado para móviles, tablets y escritorio.
- **Optimizado para SEO**: Incluye Schema.org y metaetiquetas para buscadores.
- **Integración con WhatsApp**: Captura de leads directa al número de soporte.

## 🛠️ Tecnologías
- HTML5 (Semántico)
- CSS3 (Vanilla, Glassmorphism, Animaciones)
- JavaScript (Vanilla)
- FontAwesome 6

## 📸 Captura del Hero
El nuevo diseño cuenta con un fondo inspirado en la fibra óptica y una tipografía moderna con gradientes.
